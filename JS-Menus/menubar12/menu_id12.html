Explanation


Here's another example of Simple Cascading Style Sheet. This will, work with Browser Versions 4.0 (or later). On older Browsers, the Styled Links will appear as only Text Links.

At first look you may think, these are "Button" Images. Well, let me tell you, these aren't Images. So what have I done here? I have specified a Style - if you download & take a look at the script, you'll find that I have done the following:
  • I have first created a "Style Specification" which will apply to all <h4> tags. This helps us to use the style all through rather than having to Define the same Style repeatedly.
  • In the Style Sheet, I have asked for Light Blue Background, Green Outset Border (to give it a 3D look) and a Cell padding.
  • I have then enclosed all link descriptions within "h4" tags.
