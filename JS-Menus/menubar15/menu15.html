To start with, I must clarify that I am not suggesting something like this one should be your "Main Menubar" unless of course, you do not wish to have a JavaScript or a DHTML Menu on your page - for whatever reason, such as your intention to have a quick loading page or may be you already have too many graphics on your page or you have a huge list of links....

So let's assume this is your "alternate/secondary" menubar - the likes that you see at the bottom of Web Pages. Then again, you can have this as your main menu, if you wish. Either way, it could/would look like the one shown above. And it's real easy to construct.
  • First I have instructed the browser that no TEXT at "a" tags should be underlined through statement "text-decoration:none;" and that FONT should be Tahoma, failing which Verdana of 13px size.
  • Then that the link color should be Red, for which I've used standard RGB code.
  • Finally that when the mouse pointer hovers over the links it should have a Red background with Light Lemon text.

Note:

If you are using Netscape (versions 3 or 4) & wondering what's this all about - well, I'm sorry that this won't work on anything below Netscape6. That is, there is no special effect except that, if you have noticed, unlike standard HTML pages, these "links" are not UNDERLINED. Links are still there & the Netscape browser will not crash or anything like that. On the other hand, if you are on IE 4+, you'll probably like what you see.
